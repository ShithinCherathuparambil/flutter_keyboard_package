## 0.0.1

* Initial stable release of `future_keyboard_kit`.
* Added customizable `FlutterKeyboard` with built-in layouts:
  numeric, decimal, alphanumeric, alphabetic, email, url, phone, hex, calculator, otp, date, time, currency, and scientific.
* Added `FutureKeyboard` with live theme switching, snippets, predictions, and interactive key sizing.
* Added `CalculatorKeyboard` with expression evaluation and error callbacks.
* Added comprehensive widget and integration test coverage in the package and example app.
